# realtime-deprem-API

Yayincilarin OBS'de kullanabilecegi bir API tasarladim ayni bu projenin icinde bir Client olusturdum canliya aldigim url ile birlikte
OBS'de tarayici eklentisine ekleyerek kullanabilirler

Nasil Calisir?
Eklenen tarayici eklentisi pasif olarak ekranda herhangi bir gostermeden bekler gelen veriyi ben aliyorum isliyorum degerlendirip
depremin buyuklugune gore client'e veriyi gonderiyorum ve bildirim bastiriyorum.

Bu projemi yayincilar icin tasarladim hem projeyi yaparken gercek zamanli api tasarlarken tecrubem oldu ve yayinciyi izleyen onlarca insanlarin aninda kirli bilgi olmadan dogruya yakin bilgiyi direkt olarak gormesi ve bilgilenmesini amacliyorum umarim herkes begenir gelirstirmeye devam edecegim.
